About Project

🏠 House Rent Prediction System

This project is a Machine Learning-based web application that predicts house rent based on various features such as BHK, size, location (city), furnishing status, floor, and number of bathrooms.

The system uses a trained Random Forest model to analyze historical housing data and provide accurate rent predictions. Users simply enter property details, and the model instantly estimates the expected rent.

This project is built using Python, Flask, HTML, and CSS. It is designed to be fast, user-friendly, and helpful for tenants, property owners, and real estate analysis.
